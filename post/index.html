Hello World

Welcome! This is a fully static personal homepage: content on the left, comments on the right, and an interactive terminal below.

Try: ls, cd post, cd aboutme, cat index.html, theme light.

Command Cheat Sheet
Command Args Description
ls None List directories or files in the current level.
cd [path/category] Enter the specified article category.
.. None Go back to the parent directory.
cat [filename] Open and read the specified article.
say [text] Leave a comment or message.
theme [dark|light] Switch theme and save preference.
help None Show all available commands and their descriptions.
clear None Clear the terminal screen.
